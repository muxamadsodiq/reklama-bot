REJA8 — AI-POWERED SHABLON TO'LDIRISH (Groq + per-channel templates)
=====================================================================

MAQSAD
------
Har kanal uchun admin o'z shablonini belgilaydi (masalan: Nomi, Narxi,
Holati, Manzil, Telefon, Tavsif). Foydalanuvchi e'lon yozganda bot Groq
AI orqali matnni shu shablonga avtomatik ajratadi (kiril→lotin, tartibsiz
matn → tuzilgan maydonlar), tap-to-copy qilib chiqaradi, majburiy bo'sh
maydonlarni alohida so'raydi, har maydonni ✏️ bilan tahrirlash mumkin.

TANLANGAN ARXITEKTURA
---------------------
1. AI provayder: Groq API
   - Primary: llama-3.3-70b-versatile (1K req/kun, 100K tok/kun)
   - Fallback: llama-3.1-8b-instant (14.4K req/kun, 500K tok/kun)
   - Auto-fallback rate limit/429 bo'lsa
2. Template storage: per-channel DB jadval
3. Default shablon (agar admin yaratmasa):
   Nomi*, Narxi*, Holati (Yangi/Ishlatilgan), Manzil, Telefon*, Tavsif
4. Permissions: kanal owner + super admin tahrirlay oladi
5. Parser pipeline:
   a) Regex urinish: "Maydon: qiymat" shaklini izlash
   b) 50%+ topilsa → AI faqat tozalash (kirill→lotin, format)
   c) Regex yetmasa → butun matnni AI JSON ajratadi
   d) Majburiy maydon bo'sh → user'dan alohida so'rash
   e) Har maydon yonida ✏️ tahrirlash tugmasi
6. UI: <pre> HTML bilan tap-to-copy ko'rsatiladi

DB SCHEMA
---------
CREATE TABLE channel_templates (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  channel_id INTEGER NOT NULL,
  field_key TEXT NOT NULL,         -- 'name', 'price', 'phone'...
  label TEXT NOT NULL,             -- 'Nomi', 'Narxi'...
  type TEXT DEFAULT 'text',        -- text | number | choice | phone
  required INTEGER DEFAULT 0,
  position INTEGER DEFAULT 0,
  options_json TEXT,               -- ["Yangi","Ishlatilgan"] uchun
  UNIQUE(channel_id, field_key)
);

Seed: default 6 maydon yangi kanal qo'shilganda avtomatik
yaratiladi (yoki migratsiyada mavjud kanallarga).

FALLBACK VA XATOLAR
-------------------
- Groq 429/rate limit → llama-3.1-8b-instant
- Ikkala model xato → regex natijasini ko'rsatish (yarim to'liq)
- To'liq xato → user'ga "AI hozir ishlamayapti, oddiy shaklda yozing:
  Nomi: ... Narxi: ..." deb xabar
- Timeout 15 sek → cancel, user'ga "qayta urinish" tugmasi

CALLBACK NAMES
--------------
- tmpl:list:<channel_id>      — shablon ko'rish
- tmpl:add:<channel_id>       — maydon qo'shish
- tmpl:edit:<field_id>        — tahrirlash
- tmpl:del:<field_id>         — o'chirish
- tmpl:reset:<channel_id>     — default'ga
- u:ai:edit:<field_key>       — user AI natijasini tahrirlash
- u:ai:confirm                — tasdiqlash
- u:ai:retry                  — qayta tahlil
